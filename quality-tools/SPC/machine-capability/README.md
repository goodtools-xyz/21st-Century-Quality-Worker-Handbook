# 机器能力研究

## 概念简介

机器能力研究（Machine Capability Study，MCS）是一种评估设备或机器在稳定、受控条件下生产合格产品能力的方法。它关注的是机器自身的变异，不考虑操作者、材料、环境等其他因素的影响。机器能力研究是质量管理体系中的重要组成部分，通常在以下场景中进行：

- 新设备验收阶段
- 设备大修后的能力验证
- 设备调整或重新校准后
- 作为过程能力研究的前期准备
- 质量问题分析过程中

机器能力研究的核心是计算机器能力指数（Cm和Cmk），通过这些指数可以量化评估机器满足规格要求的程度。

## 参考标准

机器能力研究通常参考以下标准：

- ISO 22514-3：统计方法应用 - 过程能力研究 - 第3部分：机器能力研究
- VDA Volume 4：系统供应商质量能力 - 第4部分：过程能力分析
- AIAG SPC手册

## 机器能力指数

### 1. Cm（机器能力指数）

Cm指数衡量机器的固有变异相对于规格公差范围的比例，不考虑过程均值的偏移。计算公式如下：

$Cm = \frac{USL - LSL}{6\sigma_{m}}$ 或 $Cm = \frac{T}{6\sigma_{m}}$ 

其中：
- USL为规格上限
- LSL为规格下限
- T为规格公差范围（T = USL - LSL）
- $\sigma_{m}$为机器的固有标准差（短期标准差）

### 2. Cmk（机器能力指数，考虑偏移）

Cmk指数考虑了过程均值可能偏离目标值的情况，是更严格的机器能力评估指标。计算公式如下：

$Cmk = \frac{\min(USL - \bar{X}, \bar{X} - LSL)}{3\sigma_{m}}$ 

其中：
- $\bar{X}$为过程均值
- 其他符号含义同Cm

## 机器能力评判标准

根据VDA Volume 4和ISO标准，机器能力的评判标准如下：

| 指数值 | 评判结果 | 建议措施 |
|-------|---------|--------|
| Cm ≥ 2.0<br>Cmk ≥ 1.67 | 能力充足 | 可以投入生产，定期监控 |
| 1.67 > Cm ≥ 1.33<br>1.33 > Cmk ≥ 1.00 | 能力尚可 | 需监控生产过程，考虑优化措施 |
| 1.33 > Cm ≥ 1.00<br>1.00 > Cmk ≥ 0.67 | 能力不足 | 必须采取改进措施，生产时加强检验 |
| Cm < 1.00<br>Cmk < 0.67 | 能力严重不足 | 不能投入生产，必须彻底改进机器或调整工艺 |

## 机器能力研究的实施步骤

### 1. 准备阶段

- **确定研究目的**：明确研究的具体目标和范围
- **选择关键特性**：选择需要评估的关键质量特性（CTQ）
- **确定规格要求**：获取并确认所选特性的规格上下限
- **准备测量系统**：确保测量系统经过验证（MSA研究合格）
- **设备准备**：确保设备经过校准，处于最佳状态
- **材料准备**：使用稳定、一致的原材料
- **环境控制**：确保测试环境稳定（温度、湿度等）

### 2. 数据收集

- **样本量**：通常收集50个连续零件的数据（VDA推荐），或至少30个零件
- **收集方式**：在短时间内（通常1-2小时）连续生产并测量，减少外部因素影响
- **记录数据**：使用规定的表格记录所有测量数据，包括时间、设备参数等
- **数据验证**：检查数据的完整性和准确性

### 3. 数据分析

- **计算均值**：计算样本的平均值$\bar{X}$
- **计算标准差**：计算样本的标准差$\sigma_{m}$
- **绘制直方图**：观察数据分布是否符合正态分布
- **正态性检验**：如果样本量较小，可进行Shapiro-Wilk检验或Kolmogorov-Smirnov检验
- **计算Cm和Cmk**：根据公式计算机器能力指数
- **评估结果**：根据评判标准评估机器能力

### 4. 报告与改进

- **生成报告**：记录研究过程、数据和结果
- **结果分析**：分析影响机器能力的因素
- **制定改进计划**：针对不足项制定改进措施
- **实施改进**：执行改进计划
- **验证效果**：重新进行机器能力研究验证改进效果

## 标准差的计算方法

在机器能力研究中，标准差的计算通常有以下几种方法：

### 1. 样本标准差法

直接使用统计公式计算样本标准差：

$\sigma_{m} = \sqrt{\frac{\sum_{i=1}^{n}(X_i - \bar{X})^2}{n-1}}$ 

### 2. 极差法（用于分组数据）

当数据按子组收集时，可使用极差法估计标准差：

$\sigma_{m} = \frac{\bar{R}}{d_2}$ 

其中：
- $\bar{R}$为平均极差
- $d_2$为与子组大小相关的常数（可从统计常数表中查得）

## 机器能力与过程能力的区别

| 特性 | 机器能力研究 | 过程能力研究 |
|-----|------------|------------|
| 研究对象 | 仅机器自身的变异 | 整个生产过程的变异 |
| 影响因素 | 仅考虑机器因素 | 考虑人机料法环测等所有因素 |
| 数据收集时间 | 短时间内（1-2小时） | 较长时间（通常≥30个工作日） |
| 样本量 | 50件（VDA推荐） | 至少100-125个数据点 |
| 使用的指数 | Cm和Cmk | Cp、Cpk、Pp、Ppk |
| 应用阶段 | 新设备验收、设备大修后 | 生产稳定后、持续监控 |
| 目的 | 评估机器固有能力 | 评估整个过程的实际能力 |

## 应用场景

机器能力研究适用于以下情况：

1. **新设备验收**：确认新设备是否满足生产要求
2. **设备改造/大修后**：验证改造或维修后的设备性能
3. **新产品导入**：在新产品量产前评估设备是否能满足产品要求
4. **质量问题分析**：当出现质量问题时，确定是否由设备能力不足导致
5. **工艺优化**：为工艺参数调整提供依据
6. **供应商评估**：评估供应商的设备能力

## 注意事项

1. **测量系统的影响**：确保测量系统的变异相对于产品公差足够小（通常MSA变异应小于公差的10%）
2. **数据正态性**：机器能力研究通常假设数据服从正态分布，如不服从，可能需要进行数据转换或采用非参数方法
3. **短期与长期变异**：机器能力反映的是短期变异，不代表长期生产中的实际表现
4. **样本代表性**：确保样本能代表机器的正常工作状态
5. **避免过度调整**：在研究过程中避免对设备进行不必要的调整
6. **持续监控**：即使机器能力满足要求，也应在生产过程中持续监控
7. **综合分析**：机器能力研究结果应与其他质量工具结合使用，如控制图、FMEA等

## 实例说明

### 机器能力研究示例

假设某机加工设备生产的零件直径规格为20.00 ± 0.10 mm（即USL=20.10，LSL=19.90），在短时间内连续生产50个零件并测量直径，获得以下统计结果：

- 样本均值$\bar{X}$ = 20.02 mm
- 样本标准差$\sigma_{m}$ = 0.015 mm

**计算机器能力指数**：

1. 计算Cm：
   $Cm = \frac{20.10 - 19.90}{6 \times 0.015} = \frac{0.20}{0.09} = 2.22$ 

2. 计算Cmk：
   - 上侧能力：$USL - \bar{X} = 20.10 - 20.02 = 0.08$ 
   - 下侧能力：$\bar{X} - LSL = 20.02 - 19.90 = 0.12$ 
   - 取最小值：0.08 
   - $Cmk = \frac{0.08}{3 \times 0.015} = \frac{0.08}{0.045} = 1.78$ 

**评估结果**：
Cm = 2.22 > 2.0，Cmk = 1.78 > 1.67，因此判定该机器能力充足，可以投入生产。

通过以上示例可以看出，即使过程均值存在轻微偏移（20.02 vs 目标值20.00），机器仍然具备足够的能力满足规格要求。

正确实施机器能力研究对于确保设备满足生产要求、预防质量问题、提高生产效率具有重要意义，是质量管理中的一项基础性工作。
