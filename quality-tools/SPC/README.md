# SPC - 统计过程控制

## 概念简介

统计过程控制（Statistical Process Control，SPC）是一种通过应用统计技术来监控和控制过程变异的方法。它使用控制图等统计工具来识别过程中的异常波动，区分过程的普通原因变异和特殊原因变异，从而及时采取纠正措施，保持过程的稳定性和可预测性。

## 参考标准

- ISO 16269-4-2010：数据统计分析 - 第4部分：检测离群值的方法

## 应用场景

- 生产过程的实时监控
- 新产品导入阶段的过程稳定性验证
- 供应商质量控制
- 质量问题的根本原因分析
- 持续改进项目中的过程优化

## 实施步骤

1. 选择需要监控的关键特性
2. 确定测量方法和样本计划
3. 收集初始数据并计算控制界限
4. 建立控制图进行监控
5. 分析异常并采取纠正措施
6. 定期评审和更新控制图
7. 进行过程能力分析

SPC是质量管理中的核心工具之一，通过科学的统计方法识别过程变异，帮助组织实现预防为主的质量理念，减少浪费，提高产品质量和生产效率。
