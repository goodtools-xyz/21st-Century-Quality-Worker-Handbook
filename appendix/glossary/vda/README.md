# VDA 基础术语

本文档汇总了德国汽车工业联合会（VDA - Verein der Automobilindustrie）系列标准中的核心术语和定义，旨在为汽车行业质量工作者提供统一的术语理解和参考。所有术语定义均参考官方标准文件，确保术语的准确性和权威性。

## 1. VDA 标准体系概述

### 1.1 VDA 标准系列术语

|------|------|------|
| VDA | Verein der Automobilindustrie | 德国汽车工业联合会，代表德国汽车制造商和供应商的行业组织 |
| VDA 标准 | VDA Standards | 德国汽车工业联合会发布的一系列质量管理、审核和改进标准 |
| VDA 6.1 | VDA Volume 6, Part 1 | 德国汽车工业联合会发布的质量管理体系审核标准 |
| VDA 6.2 | VDA Volume 6, Part 2 | 德国汽车工业联合会发布的服务质量体系审核标准 |
| VDA 6.3 | VDA Volume 6, Part 3 | 德国汽车工业联合会发布的过程审核标准 |
| VDA 6.4 | VDA Volume 6, Part 4 | 德国汽车工业联合会发布的产品审核标准 |
| VDA 6.5 | VDA Volume 6, Part 5 | 德国汽车工业联合会发布的项目管理审核标准 |

| VDA 2 | VDA Volume 2 | 德国汽车工业联合会发布的生产材料标签和包装标准 |
| VDA 9001 | VDA Implementation of ISO 9001 | 德国汽车工业联合会对ISO 9001的解释和补充 |

### 1.2 质量管理基础术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 质量管理 | Quality Management | 指导和控制组织关于质量的协调活动 |
| 质量保证 | Quality Assurance | 质量管理的一部分，致力于提供质量要求会得到满足的信任 |
| 质量控制 | Quality Control | 质量管理的一部分，致力于满足质量要求 |
| 质量改进 | Quality Improvement | 质量管理的一部分，致力于增强满足质量要求的能力 |
| 质量策划 | Quality Planning | 质量管理的一部分，致力于设定质量目标并规定必要的运行过程和相关资源以实现质量目标 |
| 质量方针 | Quality Policy | 由组织的最高管理者正式发布的该组织总的质量宗旨和方向 |
| 质量目标 | Quality Objectives | 关于质量的可量化的目标，与质量方针保持一致，可分解到组织的相关职能和层次 |

## 2. VDA 6.3 过程审核术语

### 2.1 审核基础术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 过程审核 | Process Audit | 对过程进行系统的、独立的并形成文件的检查，以确定过程是否符合要求 |
| 审核员 | Auditor | 实施审核的人员，需经过专业培训和授权 |
| 审核组长 | Audit Team Leader | 负责组织和领导审核的人员 |
| 审核计划 | Audit Plan | 描述审核的目的、范围、准则、资源和日程安排的文件 |
| 审核范围 | Audit Scope | 审核的内容和界限，包括地理位置、组织单元、活动和过程以及审核所覆盖的时期 |
| 审核准则 | Audit Criteria | 用作依据的一组方针、程序或要求 |
| 审核证据 | Audit Evidence | 与审核准则有关的并能够证实的记录、事实陈述或其他信息 |
| 审核发现 | Audit Finding | 将收集到的审核证据对照审核准则进行评价的结果 |
| 审核结论 | Audit Conclusion | 审核组考虑了审核目的和所有审核发现后得出的最终审核结果 |
| 审核报告 | Audit Report | 描述审核结果的正式文件 |

### 2.2 过程审核评分术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 符合率 | Conformity Rate | 符合要求的项目数占总项目数的百分比 |
| 评分 | Rating | 对过程或要素符合要求程度的评价，通常采用1-5分制 |
| 加权分数 | Weighted Score | 考虑要素重要性的加权评分 |
| 总体评分 | Overall Rating | 过程审核的最终评分，反映过程的总体符合性 |
| 风险分析 | Risk Analysis | 评估不符合项对产品质量和过程能力影响的系统性方法 |
| 优先顺序 | Priority | 基于风险分析确定的不符合项整改优先顺序 |

### 2.3 过程审核要素术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 人员/素质 | Personnel/Qualification | 与过程相关的人员能力和培训情况 |
| 设备/设施 | Equipment/Facilities | 用于过程的设备和设施的适用性和维护状况 |
| 工作环境 | Work Environment | 影响过程的物理、社会、心理和环境因素 |
| 材料/零部件 | Materials/Components | 用于过程的原材料和零部件的质量和管理 |
| 方法/指导书 | Methods/Instructions | 过程执行的方法和指导文件 |
| 测量/监控 | Measurement/Monitoring | 过程和产品的测量和监控活动 |
| 运输/搬运 | Transportation/Handling | 产品和材料的运输和搬运过程 |
| 包装/标识 | Packaging/Identification | 产品的包装和标识活动 |

## 3. VDA 6.4 产品审核术语

### 3.1 产品审核基础术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 产品审核 | Product Audit | 对产品进行系统的、独立的并形成文件的检查，以确定产品是否符合要求 |
| 抽样 | Sampling | 从总体中抽取样本的过程 |
| 样本 | Sample | 从总体中抽取的用于检验或测试的产品单位 |
| 检查特性 | Inspection Characteristic | 需要检查的产品特性或要求 |
| 缺陷 | Defect | 不满足与预期或规定用途有关的要求 |
| 严重度 | Severity | 缺陷对产品功能和使用的影响程度 |
| 缺陷分类 | Defect Classification | 根据严重程度对缺陷进行的分类，如A类（严重）、B类（主要）、C类（次要） |
| 产品特性 | Product Characteristic | 产品的可区分特征 |
| 功能特性 | Functional Characteristic | 与产品功能相关的特性 |
| 尺寸特性 | Dimensional Characteristic | 与产品尺寸相关的特性 |
| 外观特性 | Visual Characteristic | 与产品外观相关的特性 |
| 材料特性 | Material Characteristic | 与产品材料相关的特性 |

### 3.2 产品审核评分术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 质量特性指数 | Quality Characteristic Index | 衡量产品特性符合要求程度的指标 |
| 质量指数 | Quality Index | 衡量产品整体质量的指标，通常基于缺陷数和严重度计算 |
| A类缺陷 | Category A Defect | 严重影响产品安全性或法规符合性的缺陷 |
| B类缺陷 | Category B Defect | 严重影响产品功能或使用寿命的缺陷 |
| C类缺陷 | Category C Defect | 轻微影响产品功能或外观的缺陷 |
| 合格产品 | Conforming Product | 满足所有规定要求的产品 |
| 不合格产品 | Nonconforming Product | 不满足一项或多项规定要求的产品 |

## 4. VDA 2 物流与标签术语

### 4.1 包装术语

| 术语 | 英文 | 定义 |

| 术语 | 英文 | 定义 |
|------|------|------|

| JIT配送 | Just-In-Time Delivery | 准时配送，在需要的时间提供需要的产品 |
| JIS配送 | Just-In-Sequence Delivery | 按序配送，按照生产顺序提供产品 |

| 循环取货 | Milk Run | 定时、定路线的物料收集和配送方式 |
| 包装标准化 | Packaging Standardization | 统一包装尺寸和规格的活动 |

## 5. VDA 4 供应商管理术语

### 5.1 供应商选择术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 供应商选择 | Supplier Selection | 确定和评估潜在供应商并选择最适合的供应商的过程 |
| 供应商评估 | Supplier Evaluation | 对供应商能力和表现进行的系统性评估 |
| 潜在供应商 | Potential Supplier | 可能成为组织供应商的企业 |
| 合格供应商 | Approved Supplier | 已通过评估并被认可的供应商 |
| 供应商评级 | Supplier Rating | 基于供应商表现对其进行的等级评定 |
| 供应商发展 | Supplier Development | 帮助供应商提高能力和表现的活动 |
| 供应商审核 | Supplier Audit | 对供应商质量管理体系、过程或产品进行的审核 |

### 5.2 供应商管理术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 供应商质量管理 | Supplier Quality Management | 对供应商进行质量控制和改进的活动 |
| 供应商关系管理 | Supplier Relationship Management | 管理与供应商关系的策略和活动 |
| 供应商绩效 | Supplier Performance | 供应商满足要求的程度 |
| 早期供应商参与 | Early Supplier Involvement (ESI) | 在产品开发早期阶段让供应商参与的策略 |
| 供应商会议 | Supplier Meeting | 与供应商讨论质量、交付和绩效的会议 |
| 供应商改进计划 | Supplier Improvement Plan | 针对供应商表现不佳而制定的改进计划 |

## 6. VDA 6.5 项目管理术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 项目 | Project | 为创造独特的产品、服务或成果而进行的临时性工作 |
| 项目管理 | Project Management | 对项目进行规划、执行、监控和收尾的过程 |
| 项目经理 | Project Manager | 负责领导项目团队实现项目目标的人员 |
| 项目团队 | Project Team | 参与项目执行的人员 |
| 项目计划 | Project Plan | 描述项目目标、范围、时间、资源和风险的文件 |
| 里程碑 | Milestone | 项目中的重要时间点或事件 |
| 风险 | Risk | 不确定性对项目目标的影响 |
| 风险管理 | Risk Management | 识别、评估和应对风险的过程 |
| 项目审核 | Project Audit | 对项目进行的系统性检查 |
| 项目阶段 | Project Phase | 项目生命周期中的一个阶段 |

## 8. VDA 质量管理工具术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 因果图 | Cause and Effect Diagram | 用于识别问题可能原因的图形工具，也称为鱼骨图 |
| 5W1H | 5W1H Analysis | 一种问题分析方法，包括What（什么）、Why（为什么）、Where（哪里）、When（何时）、Who（谁）和How（如何） |
| 5个为什么 | 5 Whys | 通过连续问"为什么"来识别问题根本原因的方法 |
| 检查表 | Check Sheet | 用于系统收集和记录数据的表格 |
| 直方图 | Histogram | 用于展示数据分布的条形图 |
| 散布图 | Scatter Diagram | 用于展示两个变量之间关系的图形 |
| 排列图 | Pareto Chart | 用于识别问题主要原因的图形工具，基于帕累托原理 |
| 控制图 | Control Chart | 用于确定过程是否稳定或是否具有可预测的性能的图形工具 |
| 过程能力分析 | Process Capability Analysis | 评估过程满足规格要求能力的方法 |
| 失效模式及影响分析 | Failure Mode and Effects Analysis (FMEA) | 一种系统性方法，用于识别产品或过程中潜在的失效模式及其原因和影响 |
