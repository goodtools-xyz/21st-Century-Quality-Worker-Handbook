# 质量工具与方法术语

本页面包含质量管理中常用的工具与方法术语的定义，这些定义参考了最新的质量管理标准。

| 术语 | 缩写 | 定义 |
|------|------|------|
| 六西格玛 | Six Sigma | 一种以数据为基础的质量管理方法，旨在通过减少变异和缺陷来提高质量，目标是在100万次机会中不超过3.4个缺陷（DPMO） |
| 全面质量管理 | TQM | 一个组织以质量为中心，以全员参与为基础，目的在于通过顾客满意和本组织所有成员及社会受益而达到长期成功的管理途径 |
| 统计过程控制 | SPC | 应用统计技术对过程中的各个阶段进行监控，从而达到改进与保证质量的目的，是一种预防性的质量控制方法 |
| 测量系统分析 | MSA | 对测量系统的变异进行分析和评估的活动，确保测量数据的准确性和可靠性 |
| 失效模式与影响分析 | FMEA | 一种系统化的工程技术，用于识别产品或过程中潜在的失效模式，并评估其可能的影响，以便采取预防措施 |
| 控制计划 | Control Plan | 对控制产品所要求的系统和过程的形成文件的描述，包括控制特性、控制方法和验收标准 |
| 潜在失效模式与影响分析 | PFMEA | 用于分析过程中潜在的失效模式及其对产品质量或过程性能的影响 |
| 设计失效模式与影响分析 | DFMEA | 用于分析产品设计中潜在的失效模式及其对产品功能和性能的影响 |
| 过程能力指数 | Cpk | 衡量过程能力满足规格要求的程度的指标，考虑了过程均值与规格中心的偏移 |
| 因果图 | Cause and Effect Diagram | 一种用于识别问题根本原因的工具，也称为鱼刺图或石川图 |
| 帕累托图 | Pareto Chart | 一种用于识别和优先解决主要问题的图表工具，基于帕累托原则（80/20法则） |
| 控制图 | Control Chart | 一种用于监控过程变异的统计工具，通过设置控制限来判断过程是否稳定 |
| 检查表 | Check Sheet | 一种用于系统收集和记录数据的表格工具，便于后续分析 |
| 直方图 | Histogram | 一种用于展示数据分布情况的图表工具，显示数据在不同区间的频率 |
| 散点图 | Scatter Diagram | 一种用于展示两个变量之间关系的图表工具，帮助识别变量间的相关性 |
| 流程图 | Flow Chart | 一种用于描述过程步骤和顺序的图表工具，帮助理解和改进过程 |
