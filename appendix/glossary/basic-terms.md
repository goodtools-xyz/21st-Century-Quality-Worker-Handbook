# 质量管理基础术语

本页面包含根据最新ISO 9000:2015标准定义的质量管理基础术语。

| 术语 | 缩写 | 定义 |
|------|------|------|
| 质量 | Quality | 客体的一组固有特性满足要求的程度（ISO 9000:2015 3.6.2） |
| 质量管理 | QM | 关于质量的管理，包括确定质量方针、质量目标、质量策划、质量控制、质量保证和质量改进（ISO 9000:2015 3.2.8） |
| 质量方针 | Quality Policy | 由组织的最高管理者正式发布的关于质量的组织意图和方向的声明（ISO 9000:2015 3.5.4） |
| 质量目标 | Quality Objectives | 关于质量的、在质量方针框架内规定的目标，可量化且与质量方针保持一致（ISO 9000:2015 3.7.1） |
| 质量保证 | QA | 质量管理的一部分，致力于提供质量要求会得到满足的信任（ISO 9000:2015 3.2.11） |
| 质量控制 | QC | 质量管理的一部分，致力于满足质量要求（ISO 9000:2015 3.2.10） |
| 质量改进 | Quality Improvement | 质量管理的一部分，致力于增强满足质量要求的能力（ISO 9000:2015 3.2.12） |
| 质量策划 | Quality Planning | 质量管理的一部分，致力于制定质量目标并规定必要的运行过程和相关资源以实现质量目标（ISO 9000:2015 3.3.5） |
| 质量管理体系 | QMS | 组织建立质量方针和质量目标以及实现这些目标的过程的相互作用的一组要素（ISO 9000:2015 3.5.1） |
| 质量手册 | Quality Manual | 规定组织质量管理体系的文件（ISO 9000:2015 3.8.2） |
| 过程 | Process | 利用输入实现预期结果的相互关联或相互作用的一组活动（ISO 9000:2015 3.4.1） |
| 产品 | Product | 在组织和顾客之间未发生任何交易的情况下，组织能够产生的输出（ISO 9000:2015 3.7.6） |
| 服务 | Service | 至少有一项活动必须在组织和顾客之间进行的输出（ISO 9000:2015 3.7.7） |
| 顾客 | Customer | 能够或实际接受为其提供的，或应其要求提供的产品或服务的个人或组织（ISO 9000:2015 3.2.4） |
| 外部供应商 | External Supplier | 提供产品或服务的外部组织（ISO 9000:2015 3.2.5） |
| 缺陷 | Defect | 未满足与预期或规定用途有关的要求（ISO 9000:2015 3.6.9） |
| 不合格 | Nonconformity | 未满足要求（ISO 9000:2015 3.6.2） |
| 纠正 | Correction | 为消除已发现的不合格所采取的措施（ISO 9000:2015 3.6.10） |
| 纠正措施 | Corrective Action | 为消除已发现的不合格或其他不期望情况的原因所采取的措施（ISO 9000:2015 3.6.11） |
| 预防措施 | Preventive Action | 为消除潜在不合格或其他潜在不期望情况的原因所采取的措施（ISO 9000:2015 3.6.12） |
| 持续改进 | Continuous Improvement | 提高绩效的循环活动（ISO 9000:2015 3.2.13） |
| 组织 | Organization | 为实现目标而相互关联或相互作用的一组人员和设施（ISO 9000:2015 3.2.1） |
| 相关方 | Interested Party | 可影响决策或活动，被决策或活动所影响，或自认为受决策或活动影响的个人或组织（ISO 9000:2015 3.2.3） |
| 要求 | Requirement | 明示的、通常隐含的或必须履行的需求或期望（ISO 9000:2015 3.6.4） |
| 能力 | Competence | 运用知识和技能实现预期结果的本领（ISO 9000:2015 3.10.4） |
| 绩效 | Performance | 可测量的结果（ISO 9000:2015 3.7.8） |
| 风险 | Risk | 不确定性的影响（ISO 9000:2015 3.7.9） |
