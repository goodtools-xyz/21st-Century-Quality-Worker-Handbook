# 质量审核与认证术语

本页面包含根据最新ISO 9000:2015标准定义的质量审核与认证相关术语。

| 术语 | 缩写 | 定义 |
|------|------|------|
| 审核 | Audit | 为获得客观证据并对其进行客观的评价，以确定满足审核准则的程度所进行的系统的、独立的并形成文件的过程（ISO 9000:2015 3.13.1） |
| 认证 | Certification | 第三方依据程序对产品、过程或服务符合规定的要求给予书面保证（ISO 9000:2015 3.3.6） |
| 标准化 | Standardization | 为了在一定范围内获得最佳秩序，对现实问题或潜在问题制定共同使用和重复使用的条款以及编制、发布和应用文件的活动（ISO 9000:2015 3.3.10） |
| 管理评审 | Management Review | 由最高管理者对质量管理体系的有效性和适宜性进行的正式评价（ISO 9000:2015 3.13.7） |
| 内部审核 | Internal Audit | 由组织内部人员进行的审核，也称第一方审核（ISO 9000:2015 3.13.2） |
| 外部审核 | External Audit | 由组织外部人员进行的审核，包括第二方审核和第三方审核（ISO 9000:2015 3.13.3） |
| 第二方审核 | Second Party Audit | 由相关方（如顾客）或其他人员以相关方的名义进行的审核（ISO 9000:2015 3.13.4） |
| 第三方审核 | Third Party Audit | 由外部独立的审核组织进行的审核，这类组织通常是经认可的，提供符合要求（如ISO 9001）的认证或注册（ISO 9000:2015 3.13.5） |
| 审核准则 | Audit Criteria | 用于与客观证据比较的一组方针、程序或要求（ISO 9000:2015 3.13.7） |
| 审核证据 | Audit Evidence | 与审核准则有关的并能够证实的记录、事实陈述或其他信息（ISO 9000:2015 3.13.6） |
| 审核发现 | Audit Findings | 将收集到的审核证据对照审核准则进行评价的结果（ISO 9000:2015 3.13.8） |
| 审核报告 | Audit Report | 阐述审核结果的正式文件（ISO 9000:2015 3.13.9） |
