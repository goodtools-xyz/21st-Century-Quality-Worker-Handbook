# 软件行业标准和协会

### CMMI Institute（能力成熟度模型集成研究所）
- **官方网址**：[https://cmmiinstitute.com/](https://cmmiinstitute.com/)
- **简介**：CMMI Institute是软件和系统工程领域的标准制定机构，其CMMI（能力成熟度模型集成）是全球广泛认可的过程改进框架，用于评估和提升组织的软件开发和系统工程能力。

### A-SPICE（汽车软件过程改进及能力评定）
- **官方网址**：[https://www.automotive-spice.com/](https://www.automotive-spice.com/)
- **简介**：A-SPICE是基于ISO/IEC 15504（SPICE）开发的汽车行业特定过程评估模型，用于评估和改进汽车电子和软件的开发过程，确保产品质量和安全性。

### ISO/IEC JTC 1/SC 7（信息技术委员会/软件工程分委员会）
- **官方网址**：[https://www.iso.org/committee/45314.html](https://www.iso.org/committee/45314.html)
- **简介**：ISO和IEC联合技术委员会的软件工程分委员会，负责制定软件工程领域的国际标准，如ISO/IEC 12207（软件生命周期过程）和ISO/IEC 15504（SPICE）。

### IEEE Computer Society（IEEE计算机学会）
- **官方网址**：[https://www.computer.org/](https://www.computer.org/)
- **简介**：IEEE计算机学会制定了众多软件工程和计算机科学领域的标准，包括软件测试、需求工程、软件质量等方面的标准。
